## Promoter Blueprint (Jonathan Courtney / AJ&Smart)

Traffic -> Holding Pattern -> Selling Event -> Loop

| Phase | Description | AI Use Case |
|---|---|---|
| Traffic | Get attention (podcasts, events, social, paid ads) | Research, content prep, performance analysis |
| Holding Pattern | Keep people in your world (newsletter, social, YouTube) | Content generation, engagement cadence |
| Selling Event | Convert to revenue (webinars, email campaigns, demos) | Campaign pages, email sequences, lead magnets |
| Loop | Non-converters go back to holding pattern | Automated nurture |

**Key insight:** CEO = Promoter. Building is 20%, promoting is 80%.
