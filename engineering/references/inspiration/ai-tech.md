## Madar Platform

- **URL:** <https://madarplatform.com/en>
- **What it is:** Saudi Arabia-focused freight logistics SaaS (TMS). Unified platform covering real-time GPS/IoT shipment tracking, driver behavior scoring, eGovernment waybill integration, route optimization, digital payments, and a freight marketplace with 15,000+ validated trucks. Built by award-winning agency VideInfra.
- **Style tags:** dark, enterprise, data-dense, dashboard-aesthetic, industrial-precision, SaaS-marketing, telemetry-as-design
- **Notable elements:**
  - Hero is a live dashboard simulation — not marketing copy, but animated UI widgets showing real-time telemetry data (speed 56km/h, temp 42°C, humidity 30%, weight 18 tons) and compliance status cards (License: Valid, Route: Optimized, Inspection: 120 days)
  - The actual platform UI IS the visual design language — no photography, no abstract shapes
  - Feature sections use a desktop/mobile DOM-duplicate pattern indicating JS-driven tab switcher or card carousel (clicking feature name swaps displayed content)

  - Full-bleed cinematic/aerial WebP background images for stat sections
  - Rotate-device overlay present — confirms landscape-only or gyroscope-driven animation sequences exist
  - Bilingual Arabic/English with separate root URL paths
  - Key figures: 16,000 users, 800 fleet providers, 1.1M shipments
- **Best for:** Enterprise SaaS platforms, logistics/fleet tech, any B2B software product where showing the actual UI as a live demo is more persuasive than explaining features in words

## ibiCash

- **URL:** <https://www.ibi.cash/>
- **What it is:** Blockchain protocol minting a cryptocurrency backed by verified forest labor — each unit generated by one hectare of forest per day (carbon sequestration, oxygen, habitat). Supply hard-capped by living forests. Invite-code gated. Live market cap: $79.45M.
- **Style tags:** dark, monochrome, cinematic, brutalist-typography, terminal-aesthetic, WebGL, scroll-scrub, experimental, exclusive, "satellite-scanning-wilderness"
- **Notable elements:**
  - Near-total monochrome — black background, white text, gray accents (`#8F8F8F`). Zero decorative color.
  - All-caps display headlines throughout (`FORESTS`, `ONCHAIN`, `REWARDING`, `CONSERVATION`)
  - **SVG corner bracket animation** — every text block framed by L-shaped SVG brackets that draw themselves in via `stroke-dashoffset` animation as content enters view. Signature design motif.
  - Chapter-based scroll narrative — sections measured in hundreds of `vh` (731vh, 716vh, 550vh per chapter). Page is a scrollable film.
  - Three.js r175 WebGL canvas as full-width background — likely a 3D forest or satellite grid visualization
  - Grid frame delimiters — faint crosshair/axis markers overlay the viewport at low opacity (0.3), "targeting reticle" feel
  - **Text scramble effect** — characters shuffle/scramble before resolving to readable text on interactive elements
  - **Custom cursor** — context-sensitive label changes by chapter (`CLICK TO SCAN THE FOREST` / `CLICK TO SCAN THE GRID`)
  - Lenis smooth scroll for inertia-based scroll-scrub
  - Loading screen with MSDF texture (signed distance field) for crisp WebGL text rendering
  - Ambient audio toggle with scramble text cycling states
  - Invite-code gate modal with three states: entering / `ACCESS GRANTED` / `ACCESS DENIED`
  - Right-side scroll progress indicator with ~40 dot items and up/down navigation
  - Live market cap data in the UI (`$79.45M / 6.10%`)
- **Best for:** Web3/crypto launches, any product that needs to feel like a classified operation or satellite terminal — also a reference for scroll-scrub chapter structure, SVG draw animations, and text scramble effects regardless of industry
