# Technical Audit: Columbus Massage Center

**URL:** https://www.columbusmassagecenter.com/
**Audit Date:** 2026-03-05
**Auditor:** Iris (OphidianAI)

---

## Summary

Columbus Massage Center runs an outdated WordPress 6.3.7 installation on GoDaddy shared hosting with PHP 7.0.33 (end-of-life since December 2018). The site has multiple broken scripts, a dead Google+ integration, mixed content warnings, broken navigation links, and relies heavily on Vagaro for booking, gift cards, products, and health intake forms. The site is a strong candidate for a full rebuild -- the WordPress installation is a liability, not an asset.

## Rebuild Implications

### What We Keep (Integrate With)

| Item | Action |
|------|--------|
| Vagaro | Must integrate. This is their business platform for booking, gift cards, products, and intake forms. Vagaro provides embeddable widgets and iframes that work on any site. |
| Google Analytics (GA4) | Set up fresh GA4 property or keep G-Y12S5E726K. Drop the dead UA property. |
| Facebook Page | Keep linking to facebook.com/ColumbusMassageCenterandSalon. Use modern embed. |
| Google Maps | Re-embed with current Maps API. |
| Domain | Keep columbusmassagecenter.com. Will need DNS changes if we move hosting. |

### What We Replace

| Item | Replacement |
|------|-------------|
| WordPress + GoDaddy Hosting | Modern framework (Next.js/Vercel or similar). Eliminates PHP/WP security burden. |
| Graphene Theme | Custom design. Current theme is generic and outdated. |
| Yoast SEO | Built-in SEO with proper meta tags, schema, sitemap. |
| Contact Form 7 | Modern form solution or direct Vagaro intake form embed. |
| Jetpack (sharing, stats, Facebook embed) | Native social links, GA4 only, modern Facebook embed. |
| WordPress Leads / Inbound Now | Already broken. Replace with nothing or simple analytics goals. |
| EasyAzon | Already appears unused. Drop it. |
| WP Testimonial Widget | Custom testimonial section with working slider. |
| Google+ Widget | Remove entirely. Dead platform. |
| jQuery dependency chain | Modern JS. No jQuery needed. |

### What We Drop

- Google+ integration (dead since 2019)
- Universal Analytics UA-42211431-1 (deprecated)
- WordPress Leads plugin (broken, unmaintained)
- EasyAzon (unused)
- Pinterest sharing (low value)
- jQuery cookie dependency chain

### Key Considerations

- **Vagaro is the critical integration.** Their entire business runs through it. Any new site must support Vagaro's widget popups (JS embeds) and iframe forms. This is straightforward -- Vagaro widgets work on any HTML page.
- **No email marketing** platform detected. This is an opportunity to add one.
- **No Facebook Pixel** installed. If they run or plan to run Facebook ads, this should be added.
- **Content is minimal.** The actual page content is light -- mostly service descriptions and staff bios. Migration would be quick.
- **The site has been neglected.** Last homepage update was 2021. PHP version is 7+ years out of date. Multiple broken integrations. This is not a site being actively maintained.
- **Fix the Staff page URL.** Navigation links to /massage-therapists but the actual page is /massagetherapists/. This is a broken link that's been live for an unknown period.
