## Rules

- Always pick the least-recently-used template within the prospect's category
- Never send the same template to two prospects in the same batch
- SEO and hybrid templates require a competitive search before writing the email
- Update this table every time a cold email is sent
- When a reply is received, increment the Replies column for that template and recalculate Reply Rate (Replies / Times Used as %)
- Update the Performance Summary after any change
- If a template has 0 replies after 10+ sends, flag it for review/replacement
