## Status Legend

| Status              | Description                                         |
| ------------------- | --------------------------------------------------- |
| New Lead            | Identified but no research done yet                 |
| Researching         | Actively gathering info, building assessment        |
| Outreach Sent       | First cold email sent                               |
| Follow-Up Sent      | One or more follow-up emails sent                   |

| Meeting Scheduled   | Discovery call or meeting booked                    |
| Proposal Sent       | Formal proposal or quote delivered                  |
| Closed Won          | Client signed, project starting                     |
| Closed Lost         | Lead declined or went cold                          |
| Inactive            | No activity, parked for future outreach             |
