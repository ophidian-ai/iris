# Client Agreements

Full template: `operations/templates/client-agreement.md`
Handoff template: `operations/templates/client-handoff.md`

## Key Terms

- **IP:** Client owns content + final assembled site (upon full payment). Developer owns underlying code structure and retains right to reuse generic patterns.
- **Revisions:** 2 rounds included. Additional rounds $75/hour.
- **Payment:** Net 15, late payments 1.5%/mo interest. Work pauses if >15 days overdue.
- **Payment split:** 50/50 (Starter/Pro), 33/33/33 (E-Commerce)
- **Attribution:** Footer "Built by OphidianAI" with logo/link by default. $500 one-time removal fee.
- **Portfolio rights:** Developer retains portfolio rights regardless of attribution opt-out.
- **Warranty:** 30 days post-launch, covers functionality only -- not third-party outages or client modifications.
- **Termination:** 14 days notice either party. Client pays proportionally for completed work. IP transfers only upon full payment.
- **Signing tool:** Dropbox Sign ($15/mo)

## Handoff Deliverables

- Live URL and launch date
- What's included in monthly plan
- How to request changes (email eric.lefler@ophidianai.com)
- Account access details (domain registrar, GA4, GBP)
- Cancellation terms (30 days notice, client keeps domain)
