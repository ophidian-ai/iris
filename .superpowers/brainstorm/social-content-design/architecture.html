System Architecture

Modular pipeline -- each box is an independent skill or script

Social Content Engine -- Pipeline Flow
TRIGGER
Bi-weekly cron (every other Sunday AM) or manual /social-content
|
ORCHESTRATOR
.claude/skills/social-content/SKILL.md (overhauled)
- Determines date range (next 2 weeks)
- Rotates pillars and industries
- Generates 7 posts with copy, hashtags, platform adaptations
- Assigns image pipeline per post
- Outputs batch JSON for downstream modules
|
IMAGE ROUTER
.claude/skills/social-image-gen/
Routes each post to the right image source:

Before/After -- Excalidraw base + Nano Banana polish
Educational/AI -- Nano Banana from text prompt
Showcase -- Pexels stock photo API
BTS/Local -- Pexels or Nano Banana depending on context
IMAGE SOURCES
Excalidraw
Programmatic .excalidraw JSON -- wireframe mockups, before/after layouts
Nano Banana 2
AI image gen via Gemini API -- polished graphics, lifestyle scenes
Pexels API
Stock photos -- local business scenes, professional contexts
|
REVIEW GATE (Sunday)
Batch saved to marketing/social-media/batches/
Images saved to marketing/social-media/images/
Eric reviews copy + images together
Approve all / Edit specific posts / Regenerate
|
PLATFORM SCHEDULER
.claude/skills/social-scheduler/
Facebook
Meta Graph API
Instagram
Meta Graph API
LinkedIn
LinkedIn API
TikTok
TikTok API
